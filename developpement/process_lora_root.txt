====== LORA ======

- ON_QUERY -> done
    * seq == expected
        > correct
        > send ACK or DATA

    * seq < expected
        > RETRANSMISSION: last sent frame

    * seq > expected
        > lost (expected - seq) frames
        > send ACK or DATA

- ON_DATA -> done
    * seq == expected
        > correct
        > deliver data to upper layer

    * seq < expected:
        > RETRANSMISSION: last sent frame

    * seq > expected
        > lost (expected - seq) frames
        > deliver data to upper layer

- ON_JOIN -> done
    * seq == 0
        > OK
        > send JOIN_RESPONSE

    * seq != 0
        > invalid
        > ignore
