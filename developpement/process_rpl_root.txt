====== RPL ======

- ON_ACK -> done
    * seq == last_sent_frame.seq
        > correct
        > can send next frame

    * seq < expected
        > incorrect
        > ignore

    * seq > expected
        > incorrect
        > ignore

- ON_JOIN-RESPONSE -> done
    * seq == 1
        > valid
        > state = ready
        > join network

    * seq != 1
        > invalid
        > ignore

- ON_DATA -> done
    * seq == expected
        > correct
        > deliver to upper layer
    
    * seq < expected
        > incorrect
        > ignore

    * seq > expected
        > lost (expected - seq) frames
